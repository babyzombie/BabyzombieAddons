# BabyzombieAddons

Hypixel SkyBlock 生活质量工具模组，[Fabric](https://fabricmc.net/) 客户端模组。

[GitHub](https://github.com/babyzombie/BabyzombieAddons) · [Gitee](https://gitee.com/Bluesky-kk/BabyzombieAddons)

## 安装

需要 [Fabric API](https://modrinth.com/mod/fabric-api)

需要 [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin) 和 [YetAnotherConfigLib](https://modrinth.com/mod/yacl) 才能打开设置页面

## 主要功能

- **通用**：检查更新、HUD 编辑器、断线自动重连、玩家模型缩放、第三人称准星等
- **地牢**：自动重开、F4 观众隐藏、静音 Storm 雷声、每日场次计数
- **Kuudra**：血量显示、阶段/眩晕计时器、路径点信标、弩炮进度、箭毒补充、Perk 商店保护
- **猎手**：物品技能计时器、Boss 信息与边框、小熊软糖提醒、血柱显示
- **挖矿**：晶核自动传送、钻头挥动抑制、冰川矿井传送
- **花园**：害虫标记、Trevor 自动接取/呼叫、点击告示牌自动转向
- **队伍**：响应聊天指令（!allinv / !p / !warp / !join / !sc / !play / !pt）
- **弹出通知**：组队邀请、好友请求、交易请求等可交互弹窗
- **事件**：Great Spook 助手、挖水果助手

## 指令

`/bza` 打开设置界面。更多指令：`/bza play`、`/bza sc`、`/bza la`、`/bza yaw`、`/bza l`、`/bza wp`。
